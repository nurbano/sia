## Requerimientos

Primero es necesario clonar el respositorio:

```console
git clone https://github.com/nurbano/sia
```
Luego hay que ir al directorio tp3:

```console
cd sia/tp3
```
Es necesario instalar primero los paquetes requeridos:

```console
pip install -r requirements.txt
```
